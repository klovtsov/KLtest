!|scenario    |Create Account on System                                                                                                              |
|accountAction|${SystemPlatform};;create;;${SystemAddress};;${FunctAcct};;${FunctAcctPwd};;${ConnectionPort};;${ManagedAccount};;${ManagedAccountPwd}|

!|scenario    |Change Account Password on System                                          |NewPwd                                         |
|accountAction|${SystemPlatform};;password;;${SystemAddress};;${FunctAcct};;${FunctAcctPwd};;${ConnectionPort};;${ManagedAccount};;@NewPwd|

!|scenario    |Login Account to System                              |ManPwd                             |
|accountAction|${SystemPlatform};;login;;${SystemAddress};;${ManagedAccount};;${ConnectionPort};;@ManPwd|

!|scenario    |Delete Account on System                                                                                        |
|accountAction|${SystemPlatform};;delete;;${SystemAddress};;${FunctAcct};;${FunctAcctPwd};;${ConnectionPort};;${ManagedAccount}|

!|scenario            |Set Connection Tab                                 |
|waitForElementPresent|id=Connection           |                          |
|click                |id=Connection           |                          |
|type;                |id=PARFuncAcct          |${FunctAcct}              |

|type;                |id=Timeout              |${SystemTimeout}          |
|type;                |id=Port                 |${ConnectionPort}         |
|click                |id=bt_Save              |                          |
|waitForResult        |System changes for ${SystemName} saved successfully|
