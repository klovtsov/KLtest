!* Suite description
A user could appear to have multiple conflicting permissions for a particular system, account, and or file.
To prevent this, TPAM implements a precedence of permissions.This suite checks a correctnes of this.

Levels       Description
1              User - Account/File
2              User - Collection of Accounts/Files
3              User -System
4              User -Collection of Systems
5              Group-Account/File
6              Group-Collection of Accounts/Files
7              Group-System
8              Group- Collection of Systems
