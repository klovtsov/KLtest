!*> Test description
1. ForceReset command does to not support manually managed accounts.
2. For the manually managed accounts ForceResetManual command is used.  It returns Password and PasswordID.
3. ManualPasswordReset still provides the final step for manually managed accounts.
*!

!define SystemName {${prefix}PerlSystem1}
!define AccountName {${prefix}PerlAccount1}
!define ForceResetResult {You must use ForceResetManual command}
!define ManagedAccount {${prefix}Manul}

!include -c ScenarioLibrary

!|script                                                                                                                                                                                |
|Add System                               |${PerlAPIparams}|             |--SystemName=${SystemName};--PlatformName=Windows;--NetworkAddress=x.x.x.x               ||successfull        |
|Add Account                              |${PerlAPIparams}|             |--SystemName=${SystemName};--AccountName=${AccountName};--Password=PAssword1;--autoFlag=M||successfull        |
|Force Reset                              |${PerlAPIparams}|             |--systemName=${SystemName};--accountName=${AccountName}                                  ||${ForceResetResult}|
|Force Reset manual password                                                                                                                                                            |
|Confirm Manual Password reset with status|fail            |and result is|Success                                                                                                       |
|Force Reset manual password                                                                                                                                                            |
|Confirm Manual Password reset with status|success         |and result is|Success                                                                                                       |

!*> CleanUp
!|script                                                                |
|Delete System|${PerlAPIparams}||--SystemName=${SystemName}||successfull|
*!
